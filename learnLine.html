#2019 前端学习路线 ##来自一名菜鸟的学习建议： ####1. 学习，特别是在线学习，是非常辛苦的事情，为了少走弯路， 所以一定要系统学习，多借鉴与前辈们总结出来的经验。 ####2. 不要相信任何说 一周掌握 css， 一周学完 js 等话术， 哪些只是教你怎么用，不讲解基本原理，不系统学，最后忘得快， 也不会举一反三，（除非你是复习） 因此，学习任何知识，都要系统坚持。 ####3. 在线学习，最忌讳的一点是只听不练。 听老师讲，基本都能听得懂，但是如果不练习，忘的很快，后面知识点一综合，就感觉无从下手了。 听三遍不如写一遍记忆深刻。 ##基本路线如下哈： ####1.先学习 基本的PC端布局。 掌握 HTML5 + CSS3 基本布局。 HTML5: https://www.bilibili.com/video/av15241731/ CSS3 : https://www.bilibili.com/video/av15269197/ ####2、 PC端品优购综合案例： https://www.bilibili.com/video/av62204227/ HTML5 和 CSS3 提高 新特性 https://www.bilibili.com/video/av59866479/ ####3、 然后学习移动端布局。 百分比（流失）布局： https://www.bilibili.com/video/av57839784/ flex布局： https://www.bilibili.com/video/av62208552/ rem布局： https://www.bilibili.com/video/av58531896/ ####4、 之后进入 JavaScript 学习 JavaScript基础： https://www.bilibili.com/video/av56623792/ JavaScript 实战 DOM BOM 操作： https://www.bilibili.com/video/av56979338/?p=9 jquery 学习： https://www.bilibili.com/video/av58740469/?p=38 之后是 JavaScript进阶高级： https://www.bilibili.com/video/av60299046/ 可以按照如下学习：ajax --- node --- vue --- 小程序 --- react #详细参照：https://www.bilibili.com/read/cv3227638/?spm_id_from=333.788.b_636f6d6d656e74.10
